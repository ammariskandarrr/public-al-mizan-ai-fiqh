# Al-Mizan AI Fiqh

An intelligent Islamic Finance assistant powered by Artificial Intelligence.

## Features

- **Shariah Finance Chatbot**: Ask questions about Islamic Finance.
- **Document Analyzer**: Analyze documents for Shariah compliance.
- **Live Consultant**: Voice-based consultation.

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Set up environment variables.
4.  Run the development server: `npm run dev`
